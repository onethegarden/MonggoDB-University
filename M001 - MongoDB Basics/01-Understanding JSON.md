<h4>JSON</h4>

- acronym for <u>JavaScript Object Notation</u>
- each field has <u>a key</u> and <u>a value</u>





<h4>JSON grammer</h4>

- keys must be surrounded in double quoutes
- string, number, boolean, null, array, object
